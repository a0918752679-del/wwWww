# 商品圖片本機上傳功能

本版後台「商品上下架」已支援直接從本地端選擇商品圖片。

## 使用方式

1. 進入後台。
2. 到「商品上下架」。
3. 在「商品圖片」欄位點選檔案。
4. 選擇 JPG、PNG、WEBP 或 GIF 圖片。
5. 點「儲存商品」。

系統會先上傳圖片，並自動將圖片路徑寫入商品資料，例如：

```txt
/uploads/products/1720000000000-abcd1234ef56.jpg
```

## 限制

- 單張圖片上限：5MB。
- 支援格式：JPG、PNG、WEBP、GIF。
- 仍保留原本 URL 輸入方式；若沒有選擇本地圖片，系統會使用欄位中的圖片 URL。
